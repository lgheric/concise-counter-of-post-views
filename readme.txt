== Description ==
Show post views at the bottom of the post and in the post list of the admin panel.


== Installation ==

Installation from within WordPress
Visit Plugins > Add New.
Search for Concise Post Views Counter
Install and activate the Concise Post Views Counter plugin.

Manual installation
Upload the entire concise-post-views-counter folder to the /wp-content/plugins/ directory.
Visit Plugins.
Activate the Concise Post Views Counter plugin.
